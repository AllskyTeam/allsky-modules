# Run a Script Module

|   |   |
| ------------ | ------------ |
| **Status**  | Experimental  |
| **Level**  | Experienced  |
| **Runs In**  | Daytime Capture, Nighttime Capture, Day to Night Transition, Night to Day Transition, Periodic jobs  |


This module runs a custom script. The script has the following options available:

|  Setting | Description  |
| ------------ | ------------ |
| File Location | The location of the script to execute. |

**NOTES:** The script can be written in any language but MUST be executable.
